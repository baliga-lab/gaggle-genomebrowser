Adds a specialized bookmark dialog for annotating transcript start and termination sites.

Bookmark files signal that they are transcript annotation with the following tag in the header:

>type: transcripts

Attributes were added to bookmarks so that we'd have somewhere to put extra properties. Originally, I used ambiguous3Prime and ambiguousOverall. SungHo suggested some much more sensible options ok3Prime, ok5Prime and okOverall. If there are old bookmark files, the old properties will be ignored. This can easily be dealt with by loading the bookmark file into an editor and searching and replacing.
