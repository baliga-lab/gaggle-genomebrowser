Wizard for importing track data

Steps for importing track data from a file:

  • Pick file from dialog or enter URL
  • Preview file and file metadata
  • Select a file type (GFF, etc.)
  • Read lines into temp table
  • Select into into a table sorted by sequence, strand, start, end
  • Get list of represented strands
  • Optionally, attempt to compute a decent position for the track
  • Adjust the position by hand
  • Adjust track visual properties
